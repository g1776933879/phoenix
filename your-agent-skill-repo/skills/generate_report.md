# 生成报表技能

## 技能名称
generate_report

## 适用场景
当用户需要查看销售数据、订单概览、业务分析报告时使用此技能。

## 调用步骤
1. 确认用户需要的报表类型（销售日报/周报/月报/自定义范围）
2. 询问日期范围（如果用户未明确提供）
3. 调用 `generate_report` 工具，传入 startDate 和 endDate
4. 将结果格式化返回给用户

## 使用示例
- "查一下上个月的销售情况"
- "生成2025年Q1的业绩报告"
- "对比这周和上周的订单数据"

## 注意事项
- 此工具需要人工审批才能执行
- 默认只返回摘要数据，详细数据需额外请求
